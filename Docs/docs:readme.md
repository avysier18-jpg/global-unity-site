# Document Storage

This folder is the designated storage location for all static files referenced in the Global Unity website.

## Folder Structure
- /docs/monetary/ (PDFs regarding Federal Reserve data)
- /docs/housing/ (Excel/PDFs regarding Housing markets)
- /docs/oversight/ (APA filings)

## Usage
Link to files in HTML using:
`<a href="docs/filename.pdf">View Document</a>`
